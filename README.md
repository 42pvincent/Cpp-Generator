# CreateClass

Create a new class
Call it with the Class name you want, and let him do the work.

Call it with :
  - Class name

Example :
./CreateClass.sh Foo

# GetSet

Create a new getter and setter.
Call it with :
  - Class name
  - Var type
  - Var name

Example :
./getset.sh FooClass int bar

# Private
Create a new private var in the class you want.

Call it with :
  - Class name
  - Var type
  - Var name


Example :
./private.sh FooClass int bar
